功能描述：
此样例演示了USB音频设备功能
USB 音频设备主要是借助 USB MISC 类实现，将其模拟成一个双通道麦克风和扬声器，设备
程序运行后，在PC端，打开设备管理器，可以看到声音、视频和游戏控制器中出现PUYA UAC
DEMO设备。

Function descriptions:
This sample demonstrates the USB audio functionality.
USB audio devices are mainly implemented with the help of the USB MISC class, 
which simulates it as a two-channel microphone and speaker device. After the 
program runs, on the PC side, open the device Manager and you will see PUYA UAC 
DEMO appear in the sound, video and game controllers.
================================================================================
测试环境：
测试用板：STK
MDK版本：5.28
IAR版本： 9.20
GCC 版本：GNU Arm Embedded Toolchain 10.3-2021.10

Test environment:
Test board: STK
MDK Version: 5.28
IAR Version: 9.20
GCC Version: GNU Arm Embedded Toolchain 10.3-2021.10
================================================================================
使用步骤：
1. 下载并运行程序；
2. 在PC电脑上打开一个设备管理器,可以看到声音、视频和游戏控制器中出现PUYA UAC 
DEMO设备。

1. Download and run the program;
2. Open a device manager on your PC and see PUYA UAC DEMO appear in the sound、
video and game controllers.
